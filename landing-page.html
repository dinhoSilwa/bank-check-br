Simples de usar

Valide bancos e gere SVGs em poucas linhas

// Importar funções
import { validarBanco, buscarBanco, gerarSVG } from 'bank-check-br';

// Validar banco por código
const banco = await validarBanco('001');
console.log(banco?.nomeExtenso); // "Banco do Brasil"

// Buscar com SVG incluído
const comSvg = await buscarBanco('341', { includeSvg: true });
console.log(comSvg?.svg); // "<svg>...</svg>"

// Gerar SVG diretamente
const svg = gerarSVG('Nubank');

API Completa

Função Descrição
validarBanco(codigo) Valida e retorna banco por código
buscarBanco(codigo) Busca banco por código de agência
buscarBancoPorISPB(ispb) Busca banco por ISPB
buscarBancosPorNome(nome) Busca bancos por nome
listarBancos() Lista todos os bancos
listarBancosComIcone() Lista bancos com ícone SVG
gerarSVG(nome) Gera SVG do banco
recarregarDados() Recarrega dados do BCB
